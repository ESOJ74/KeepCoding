En el directorio 'data' encontraréis los ficheros de datos para cuando vayáis a trabajar con 
las diferentes prácticas del curso.

Scala_Intro_1 y y Scala_Intro_2 son las prácticas relacionadas con
el aprendizaje del lenguaje de programación Scala.

En el interior del curso encontraréis
instrucciones de instalación y puesta a punto de la herramienta Apache Zeppelin
para reailzar las prácticas, con demostraciones para entornos
entorno Mac y Linux y comentarios verbalizados para Windows.

Nota: las rutas a ficheros en windows difieren de las usadas en las 
prácticas, aunque este tipo de acotaciones
están pertinentemente comentadas.
